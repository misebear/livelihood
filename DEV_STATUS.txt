■ 프로젝트 개요
  - 수급생활: 기초생활수급자를 위한 현금흐름 & 안전 자산 관리 웹앱
  - 기술 스택: Rails 8.1.2 / PostgreSQL / Hotwire (Turbo + Stimulus) / Tailwind CSS
  - 인증: Devise / 백그라운드: Sidekiq + sidekiq-cron / 공휴일: holidays gem

■ 서버 실행 방법
  ruby bin/rails server -p 3100 -b 127.0.0.1
  (포트 3000은 다른 앱이 점유 중이므로 3100 사용)

================================================================================
  알려진 이슈 & 참고사항
================================================================================

1. 포트 충돌: localhost:3000은 HiTEL 앱이 점유 → 3100 사용
2. Ruby 버전: 시스템에 3.2/3.3/3.4 설치됨, 현재 Ruby 3.4.8로 bundle install 완료
3. Puma tmp_restart: Windows에서 exec bin/rails 실패 → 비활성화 처리
4. Redis 필요: Sidekiq 사용 시 Redis 서버 실행 필요 (기본 localhost:6379)
5. 프로덕션 배포 시: ActiveRecord Encryption 키를 Rails credentials로 이전 필요
