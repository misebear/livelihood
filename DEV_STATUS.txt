📋 보듬 (Bodeum) — 복지 혜택 관리 앱
=========================================

## 🟢 현재 진행 상태: 핵심 기능 구현 완료 + UI 리디자인 완료

## ✅ 완료된 작업

### 리브랜딩 (v2.0)
- 앱 이름: 수급생활 → **보듬** (14곳 일괄 치환)
- 컬러 팔레트: 인디고 → **딥 틸(#0D9488) + 코랄(#F97316) + 크림(#FEFCE8)**
- 6곳 그래디언트 교체, 다크모드 따뜻한 스톤 톤 적용
- Stitch MCP 활용 2025 모던 디자인 레퍼런스 생성

### 혜택 목록 UI 개선
- **카테고리 탭**: flex-wrap 줄바꿈 — 12개 탭 전체 노출 (스크롤 불필요)
- **중요도순 정렬**: 기초급여 → 자산형성 → 감면·할인 → ... → 지역특화
- **📅 신청기간 배지**: 제목 옆 초록 배지로 한눈에 확인
  - 24건 주요 혜택에 신청기간 데이터 삽입 완료
- **카테고리 내 정렬**: last_synced_at DESC (최신 데이터 우선)

### 복지 혜택 자동 수집 시스템
- DB 마이그레이션: benefits 테이블 10개 컬럼 추가
- Benefit 모델: 11개 카테고리(CATEGORIES 상수) + scope + 헬퍼
- **시드 데이터 55건** (11개 카테고리)
- **PublicDataSyncService**: 공공데이터포털 4개 API 멀티 연동
  - 중앙부처복지서비스, 지자체복지서비스, 사회서비스 제공기관, 장애인편의시설
- **WelfarePortalScraperService**: 6개 복지 포털 크롤링
- **BenefitDeduplicationService**: 중복 제거 + 데이터 정제
- **Rake 태스크**: sync:benefits, sync:api, sync:scrape, sync:dedup, sync:status
- **Windows 작업 스케줄러**: LiveliBenefitSync — 매일 새벽 4시 자동 실행

### 기존 완료 기능
- Google OAuth 로그인
- 대시보드 (수급 안전선 게이지, D-Day 카드, 현금흐름)
- 혜택 목록 + 상세 + 유저 혜택 연동
- 보호자 관리 (대리 조회 포함)
- 현금흐름 이벤트 관리
- 소득인정액 자동 계산 + 추가 저축 가능 금액
